# Plugin Porter Marketplace

Automated tools for converting Claude Code plugins to Qwen CLI + MCP architecture.

## Installation

Add this marketplace to Claude Code:

```bash
claude plugin add github:YOUR_USERNAME/plugin-porter-marketplace
```

Then enable the plugin-porter plugin.

## What's Included

- **plugin-porter**: Complete plugin conversion toolkit with agents, commands, and skills for porting Claude Code plugins to Qwen CLI
